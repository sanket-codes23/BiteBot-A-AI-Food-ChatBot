# 🍽️ Bitebot - AI Food Ordering Chatbot

RoyalFeast is an AI-powered food ordering chatbot that allows users to browse the menu, place orders, remove items, view their cart, confirm orders, and track deliveries using natural language conversations.

## 🚀 Features

- 🤖 AI-powered chatbot using Dialogflow ES
- 🍕 Browse food menu by category
- 🛒 Add multiple food items to cart
- ❌ Remove items from cart
- 📋 View current cart
- ✅ Confirm and place orders
- 🚚 Track orders using Order ID
- 💻 Modern React frontend
- ⚡ FastAPI backend
- 🗄️ MySQL database integration

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sanket-codes23/BiteBot-A-AI-Food-ChatBot.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd food-chatbot-frontend
npm install
npm run dev
```
